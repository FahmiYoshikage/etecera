# Rainmeter Configuration

Rainmeter adalah aplikasi desktop customization untuk Windows.

## Instalasi

1. Download Rainmeter dari [rainmeter.net](https://www.rainmeter.net/)
2. Salin folder skin ke `Documents\Rainmeter\Skins\`
3. Refresh Rainmeter dan load skin yang diinginkan

## Struktur

```
Rainmeter/
├── Skins/          # Custom skins
└── Layouts/        # Layout configurations
```
